# Slack Reminder Workflow Concept

## Purpose

This workflow shows how People Operations teams can use Slack reminders to reduce manual follow-up during evaluation cycles.

## Workflow Example

Trigger:
- Evaluation cycle starts
- Self-review deadline is approaching
- Manager review deadline is approaching
- Evaluation is incomplete

Actions:
1. Send reminder to employee
2. Send reminder to manager
3. Notify HR if deadline is missed
4. Update tracking sheet or dashboard
5. Send final escalation reminder before cycle close

## Example Reminder Message

Hi [Name],

This is a reminder to complete your performance review by [Deadline].

Please complete:
- Self-review
- Goal progress update
- Comments for your manager

If you have already submitted it, no action is needed.

## Operational Value

This workflow reduces manual HR follow-up, improves completion rates, and creates consistency across distributed teams.

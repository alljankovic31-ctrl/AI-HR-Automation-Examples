# 30/60/90 Onboarding Plan Prompt

## Purpose

This prompt creates a structured onboarding plan for new employees in remote or hybrid teams.

## Prompt

You are a People Operations specialist.

Create a 30/60/90-day onboarding plan for a new employee.

Role:
[ROLE]

Team:
[TEAM]

Manager:
[MANAGER]

Company context:
[COMPANY CONTEXT]

Output:
1. Day 1 priorities
2. First week checklist
3. First 30 days
4. First 60 days
5. First 90 days
6. Manager check-in questions
7. Success indicators
8. Risks to monitor

Rules:
- Keep the plan practical.
- Make it role-specific.
- Include manager responsibilities.
- Include measurable success indicators.

## Use Case

Useful for onboarding, manager enablement, employee lifecycle operations, and remote team integration.

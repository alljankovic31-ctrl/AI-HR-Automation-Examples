# Performance Review Summary Prompt

## Purpose

This prompt helps HR and managers turn raw manager feedback into a structured, neutral, evidence-based performance review summary.

## Prompt

You are an HR Operations assistant.

Analyze the following manager feedback and create a structured performance review summary.

Output format:
1. Key strengths
2. Areas for improvement
3. Evidence-based examples
4. Risk indicators
5. Recommended development actions
6. Suggested manager talking points

Rules:
- Keep the tone professional and neutral.
- Do not exaggerate.
- Separate facts from interpretation.
- Avoid biased or emotional language.
- Keep the summary concise and useful for a manager 1:1 discussion.

Manager feedback:
[PASTE FEEDBACK HERE]

## Use Case

Useful for performance reviews, manager calibration, employee development discussions, and structured HR documentation.
